double LLA::getMagnitude ( void  )
virtual

overrides position vector magnitude

Returns
ECF position magnitude

For spherical coordinates, magnitude is the third component (i.e. radius) For LLA, the third component is altitude above a reference datum, so magnitude is really measured from the center of the Earth.
